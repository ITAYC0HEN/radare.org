Advanced Trainings (September 4-5)

We are happy to present our Training for the 2019 edition of r2con.

Mobile Reverse Engineering with R2frida

8 hours | Edu Novella @enovella_, Alex Soler (aka murphy) @as0ler

Combining dynamic with static analysis is the key to quickly solving many challenges when performing binary analysis. Have you ever thought about combining Radare2 with Frida? This combination has given birth to "r2frida", an IO plugin that allows you to put the power of Frida into r2land.

During this workshop, we will walk you through how to use r2frida, primarily on mobile apps. Attendees will learn about offensive mobile security, e.g how to unpack malware, bypass jailbreak protections, SSL pinning, anti-debugging or even Frida detections using Frida itself. The workshop is suitable for pentesters, reversers, malware analysts and mobile developers who want to learn more about mobile security.

Advanced Techniques for Semi-automatic Code Deobfuscation

8 hours | Tim Blazytko

In modern businesses, code obfuscation has become a vital tool to protect, for example, intellectual property against competitors. In general, it attempts to impede program understanding by making the to-be-protected program more complex.

In this workshop, we give a basic overview about common code obfuscation techniques and deobfuscation techniques. In the second part, we introduce advanced techniques such as compiler optimizations and symbolic execution and use them to break state-of-the art code protections schemes in a semi-automatic manner. The workshop is suitable for everyone who has experience in reverse engineering of x86 code and wants to deepen his/her knowledge in advanced program analysis or code obfuscation techniques.


Basic Trainings (September 4-5)

Radare2 Developing Internals

4 hours | Pancake (@trufae)

Learning how to fix bugs, add new features, write plugins or modify the behaviour of any command in r2 by modifying its source is a very important task for any hacker, not just to make it work in the way you like but also to understand better the internals and small details.

The knowledge adquired in the training can be applied to other opensource projects, so it is interesting not just to anybody that wants to contribute to the project, but also for those who want to learn better good coding practices in C.

Dissecting binaries with Cutter

4 hours | @Megabeets_ (Check Point Research), @xarkes_, @thestr4ng3r

Who said radare2 does not have a decent GUI? Cutter is The official multi-platform GUI for radare2. Its goal is making an advanced, customizable and FOSS reverse-engineering platform while keeping the user experience at mind.

In this training we will take you to a journey inside Cutter. We'll explore many features of Cutter through real-life examples and we will get our hands dirty with analyzing a malware found in the wild. You will also learn how we can script and extend Cutter with C++ and Python plugins. And last, for those of you who are radare2 junkies, don't worry - we will learn how you can use radare2 commands straight within Cutter.

Radare2 survival training

8 hours | @xvilka, @maijin, @jvoisin

Since ancient history, humanity seeks the answer to the essential properties of software systems. To date the best answer known is UNIX. In this training, we will pull the dust off the old concepts, and refit them to the modern world of reverse engineering. During these few hours, we will learn how to survive in a console jungle without even guide.

Step by step, trainers will show you the radare2 powers, both basic and hidden: from the simple disassembly and file formats analysis to the scripting and emulating, from the debugging to recording the traces and combining radare2 with other tools. We will cover core features and standard tools from the radare2 kit. In no time, everyone, who knows how to type and know the basics of reverse engineering, will be able to juggle radare2 commands in the console to impress his friends and family.
