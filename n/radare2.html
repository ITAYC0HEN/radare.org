What's radare2?

A free/libre toolchain for easing several low level tasks like forensics, reverse engineering, exploiting, debugging, ...

The toolchain is composed by a bunch of libraries (which are extended with plugins) and programs that can be automated with almost any programming language.

 

Installation

The sources are available on Github.
Each release have precompiled binaries in a separate server.

$ git clone https://github.com/radare/radare2 $ cd radare2 $ sys/install.sh

Simple Session

Radare2 can be used in many ways, a

$ rasm2 -a arm -b 32 -d `rasm2 -a arm -b 32 nop`


Package Manager

Using the integrated package manager you can easily install external plugins from different sources, most interesting ones are the native ghidra decompiler, the r2dec decompiler and the frida integration. But there are more!

$ r2pm update $ r2pm -i r2ghidra $ r2pm -i r2dec $ r2pm -i r2frida
 

Documentation

Join the Telegram / IRC channels and feel free to ask anything. In addition you can read The Book but everything in r2 is self-documented (just append the '?' char) or read the manpages/-h:
